# Как запустить код?

## Шаг 1. Клонирование репозитория
1) Откройте терминал
2) Введите команду "git clone https://github.com/dmitriysmetanin/info-search-2"
(или воспользуйтесь иными средствами для работы с git-репозиториями)

## Шаг 2. Установка зависимостей
1) Перейдите в папку проекта
2) Установите зависимости. Для этого:
2.1) Откройте терминал в папке проекта
2.2) Введите команду "pip install virtualenv"
2.3) Введите команду "virtualenv venv"
2.4) Установите библиотеки, для этого введите команду "pip install -r requirements.txt"

## Шаг 3. Запустите скрипт для формирования токенов
1) Запустите файл tokens.py

## Шаг 4. Запустите скрипт для группировки по леммам
1) Запустите файл lemmas.py
